The First Computers

If we look at a large time scale, we can say that the first computers were invented in Mesopotamia hundreds of years before Christ, such as the abacus. However, I will direct the discussion to the MARK 1 electromechanical computer invented in 1944 to perform mathematical calculations. According to the scientists, the machine allowed mathematical calculations to be done more quickly. In addition, we can mention the ENIAC (Electronic Numerical Integrator and Computer) which was built by electrical engineers and was also an electromechanical computer and allowed 2 year calculations to be done in 2 hours. Furthermore , ENIAC was widely used to calculate missile ballistics routes .

How the Internet was Born

We can say that the internet appeared in 1969 with the ARPPANET where it had the first connection between Stanford and UCLA computers. . In the early 70s, the word Internet was born, which was applied to the system of interconnected networks through the TCP and IP protocols (Transmission Control Protocol / Internet Protocol) on which they based Internet services and e-mails and in 1990 Tim Berners-Lee created the language HTML (Hypertext Transfer Protocol).

HTML and its Tags

There are many tags in HTML , but the basics and more used are :

  • html : This tag is used to inform the browser that document is HTML

  • head : This tag is used to set informations about the title of Webpage and add scripts

  • body : This tag is used to set informations about the body of Webpage . In the body we can add texts , images , links , videos etc .

  • h1 : This tag is used to insert the headers on the Webpage . In addition , we can use whole numbers until six after the letter eich .

  • p : This tag is used to determine a paragraph on the Webpage .

  • ul : This tag is used to build a unordered list on the Webpage.

  • ol : This tag is used to build a orderly list on the Webpage.

  • strong : This tag is used to make a word body.

  • i : This tag is used to make a word italicized.

  • a : This tag is very important in HTML , because with it we can add hyperlinks , images and videos on the Webpage . In summary this tag is fundamentals because it allowed the navigation on the Website.
